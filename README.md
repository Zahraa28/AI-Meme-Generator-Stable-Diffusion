# 🎭 AI Meme Generator – Stable Diffusion
## 📌 Project Overview

This project is the final project (6/6) of the Applied AI Lab program.
The goal was to build a Generative AI application that creates memes using a Stable Diffusion model, and deploy it through an interactive Streamlit web app.

This project combines AI + creativity + real-world application, showing how generative models can be used in social media and digital marketing.

### ⚙️ Features
Generate memes using text prompts
Use Stable Diffusion for text-to-image generation
Interactive Streamlit interface
Real-time image generation
Customizable prompts for creative outputs

### 🧠 Technologies Used
Python
Stable Diffusion
Hugging Face / Diffusers
Streamlit
PyTorch

### 🔄 Workflow
User enters a text prompt (e.g., meme idea)
The prompt is passed to the Stable Diffusion model
The model generates an image based on the prompt
The image is displayed in the Streamlit app

### 📚 Key Learnings
Understanding diffusion models and text-to-image generation
Prompt engineering for better outputs
Deploying AI apps with Streamlit
Building end-to-end Generative AI systems
